Optimización del Empaquetado de Productos para Reducir Costos y Mejorar la Sostenibilidad
Descripción del Proyecto
Este proyecto tiene como objetivo principal optimizar el empaquetado de productos para reducir los costos de producción y mejorar la sostenibilidad. Se busca la implementación de materiales más eficientes, ecológicos y accesibles, sin comprometer la calidad o la seguridad del empaque, alineándose con las regulaciones ambientales y las expectativas del consumidor.

Objetivos del Proyecto:
Reducir los costos de materiales de empaquetado en un 15% en un plazo de 12 meses.
Incrementar en un 20% la disponibilidad de proveedores sostenibles en 6 meses.
Aumentar la aceptación del empaquetado sostenible en un 30% en los próximos 12 meses mediante campañas de marketing.
Estructura del Proyecto
Gestión de responsabilidades: Utilizamos Trello para asignar y gestionar tareas de los diferentes equipos involucrados.
Cronograma: El cronograma del proyecto está definido en GanttProject, con un tiempo estimado de ejecución de 12 meses.
Análisis de Factores Clave: Basado en la Matriz Vester para identificar las principales variables que afectan el proyecto.
Árbol de Problemas y Objetivos: Realizado con la herramienta MICMAC para entender las causas raíz y sus soluciones potenciales.
Instrucciones de Uso
1. Gestión de Responsabilidades:
Puedes seguir el desarrollo del proyecto a través de nuestro tablero en Trello:

Accede a Trello y busca el tablero del proyecto titulado Optimización del Empaquetado.
Cada tarea está asignada a un equipo o responsable con fechas límite, descripciones y checklists detalladas.
2. Cronograma del Proyecto:
Para revisar el cronograma completo, descarga GanttProject y abre el archivo cronograma.gantt incluido en el repositorio del proyecto. Este cronograma detalla todas las fases del proyecto, desde el análisis inicial hasta la implementación de empaques optimizados.

3. Matriz de Análisis (Vester):
Puedes encontrar la Matriz Vester en el archivo Excel matriz_vester.xlsx.
Esta matriz incluye el análisis de las interdependencias entre los factores clave del proyecto (costos, sostenibilidad, aceptación del cliente, etc.).
4. Árbol de Problemas y Árbol de Objetivos:
Los gráficos de los árboles se encuentran en los archivos arbol_problemas.png y arbol_objetivos.png, disponibles en la carpeta /diagrams. Estos árboles te permiten entender visualmente las relaciones entre las causas del problema y los objetivos planteados para su resolución.

Requisitos
Trello: Herramienta para la gestión de tareas y proyectos.
GanttProject: Software para la creación y manejo de cronogramas.
Microsoft Excel o cualquier editor de hojas de cálculo compatible para visualizar la matriz de factores.
MICMAC: Software para el análisis estructural de las relaciones entre los factores.
Información Relevante
